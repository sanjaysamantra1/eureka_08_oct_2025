Web Development
===============
-Developing a Website

1. FrontEnd : User Interface ( Look & Feel )
        HTML : Content ( div , paragraph , image , table , link, form, audio, video )
        CSS :  Styling ( border, background, margin, padding, width, height )
        Bootstrap : CSS Library (twitter) (pre-defined css classes, responsiveness)
        Javascript : Behaviour 
        Angular/React : Javascript Framework (Boilerplate project , RAD , CLI , Template system)            
2. BackEnd : Develop APIs (Java, .net, Python, NodeJs, Go)
3. Database : Stores the data ( Oracle , MySql, postgress , MongoDB )
